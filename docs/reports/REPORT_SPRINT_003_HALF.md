### 5. Базовый поиск

#### Реализация:
- Поле поиска в верхней части экрана
- Фильтрация по названию и категории
- Мгновенный отклик (без debounce)
- Обновление счётчика найденных товаров

```kotlin
val filteredProducts = remember(searchQuery, products) {
    if (searchQuery.isBlank()) {
        products
    } else {
        products.filter { 
            it.title.contains(searchQuery, ignoreCase = true) ||
            it.category.contains(searchQuery, ignoreCase = true)
        }
    }
}
```

## 🚧 Известные ограничения

### Текущие:
1. **Поиск без debounce** - может быть неоптимально при большом количестве товаров
2. **Нет пагинации** - все товары загружаются сразу
3. **Избранное не сохраняется** - только локальное состояние в UI

### Планируется в следующей части спринта:
- Добавить фильтрацию по категориям
- Реализовать пагинацию для больших списков
